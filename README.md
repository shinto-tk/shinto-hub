# Shinto [![Discord Shield](https://discord.com/api/guilds/982801516417122364/widget.png)](https://discord.gg/GYqb52EFJc)
Free Roblox Script Hub designed for shooters.  
***Shinto is still in beta stage of development so expect some bugs or lack of features.***  
***Please fork if you want to modify this hub or make your own based on it.***  
Made With :heart: By ***not alex#0001***  
### Loadstring
```lua
loadstring(game:HttpGetAsync("https://raw.githubusercontent.com/shinto-tk/shinto-hub/main/Loader.lua"))()
```
